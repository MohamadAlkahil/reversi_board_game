## Description
A command-line implementation of the Reversi board game. Players can take turns playing on an 8x8 board.

## Game Rules
+ The game starts with two white and two black pieces in the center of the board
+ Black always moves first
```
  abcdefgh
a ........
b ........
c ........
d ...WB...
e ...BW...
f ........
g ........
h ........
Enter move for colour B (RowCol):
```
+ Players must place a piece that sandwiches at least one opponent piece in a straight line (horizontal, vertical, or diagonal)
```
Enter move for colour B (RowCol): cd
  abcdefgh
a ........
b ........
c ...B.... ← Black piece placed here
d ...WB...
e ...BW... ← Creates a sandwiche with the B at position ed
f ........
g ........
h ........
```
+ All sandwiched opponent pieces are flipped to the current player's color
```
Enter move for colour B (RowCol): cd
  abcdefgh
a ........
b ........
c ...B....
d ...BB...
e ...BW...
f ........
g ........
h ........
```
+ If a player has no valid moves, their turn is skipped
+ The game ends when neither player has valid moves
+ The player with the most pieces on the board wins

## Running the Game
```
cargo run
```
